# 🔑 프로젝트 1 : 데이터베이스의 Entity-Relation 설계
## ❓ 프로젝트의 목표
✔ **택배 물류 시스템의 데이터베이스**를 설계합니다. 다음과 같은 데이터를 다룹니다.
- 고객
- 택배
- 택배 배송 및 통관 이력
- 택배 결제 이력
- 택배 물류 창고 및 운송 수단

## 🚴‍♂️ 프로젝트의 주안점
**1. 추후 확장성을 염두해 두고 설계한다.**  
**2. 이상 현상이 발생하지 않도록 BCNF으로 설계한다.**
